How to setup an SRU Server

Description

This is basic instructions to setup your library SRU server.

Setup

  1. On the Integration Profile List page (Configuration Menu > General > External Systems > Integration Profiles), select Add Integration Profile. Page 1 of the External System wizard opens.

  2. Enter a unique code for the new integration type.

  3. Enter a unique name for the new integration type.

  4. Select SRU Server for Integration Type.

The Default check box is not functional for this integration type.

  1. Select Next.

  2. Configure the SRU integration profile using the parameter explanations in the table below.

Test your SRU Server URL

SRU Server URL
https://<Alma domain>/view/sru/<institution code>

Example SRU Server:
For SUNY Oswego Penfield library
